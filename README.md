## 📂 Estrutura do Repositório
A organização das pastas é feita de forma automatizada (ou guiada pelo histórico de contests):
* Os códigos são separados por **ID do Contest** e **Letra do Problema** (ex: `1700A - Optimal Path`).
* Cada submissão aceita contém o código fonte altamente otimizado para evitar *Time Limit Exceeded (TLE)*.
